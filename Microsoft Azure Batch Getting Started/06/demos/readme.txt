Scripting Batch Service Provisioning with PowerShell

Open the AzBatch.ps1 in Notepad.
Replace BATCH_ACCOUNT_NAME with the name of the batch account you will use.
Run the commands in PowerShell to provision an Azure Batch environment.

Scripting Batch Job Processing with Azure CLI

Start a new CLI command window in the Azure portal.
Use the commands in AzureCliCommands.txt to manage the Azure batch job.
